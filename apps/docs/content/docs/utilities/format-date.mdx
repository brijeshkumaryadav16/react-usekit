# formatDate

The `formatDate` utility function formats dates into readable strings with various formatting
options. It provides a simple way to display dates consistently across your application.

## Usage

```javascript
import { formatDate } from './utils/formatDate';

const date = new Date('2024-01-15T10:30:00');
const formatted = formatDate(date, 'MM/DD/YYYY');
console.log(formatted); // "01/15/2024"
```

## API Reference

### Parameters

- `date` (Date | string | number): The date to format
- `format` (string): The format pattern to use

### Returns

- `string`: The formatted date string

### Format Patterns

- `YYYY` - 4-digit year
- `MM` - 2-digit month
- `DD` - 2-digit day
- `HH` - 2-digit hour (24-hour)
- `mm` - 2-digit minute
- `ss` - 2-digit second

### Comment Timestamps

```javascript
function Comment({ comment }) {
  const getRelativeTime = date => {
    const now = new Date();
    const commentDate = new Date(date);
    const diffInHours = (now - commentDate) / (1000 * 60 * 60);

    if (diffInHours < 24) {
      return formatDate(commentDate, 'HH:mm');
    } else if (diffInHours < 168) {
      // 7 days
      return formatDate(commentDate, 'MM/DD');
    } else {
      return formatDate(commentDate, 'MM/DD/YYYY');
    }
  };

  return (
    <div className="comment">
      <div className="comment-header">
        <span className="author">{comment.author}</span>
        <span className="timestamp">{getRelativeTime(comment.createdAt)}</span>
      </div>
      <p>{comment.text}</p>
    </div>
  );
}
```

### Calendar Component

```javascript
function Calendar({ events }) {
  const [selectedDate, setSelectedDate] = useState(new Date());

  const getDayEvents = date => {
    const dateStr = formatDate(date, 'YYYY-MM-DD');
    return events.filter(event => formatDate(event.date, 'YYYY-MM-DD') === dateStr);
  };

  return (
    <div className="calendar">
      <div className="calendar-header">
        <h2>{formatDate(selectedDate, 'MM/YYYY')}</h2>
      </div>
      <div className="calendar-grid">{/* Calendar grid implementation */}</div>
      <div className="selected-date-events">
        <h3>Events for {formatDate(selectedDate, 'MM/DD/YYYY')}</h3>
        {getDayEvents(selectedDate).map(event => (
          <div key={event.id}>
            {event.title} - {formatDate(event.date, 'HH:mm')}
          </div>
        ))}
      </div>
    </div>
  );
}
```

### Form Date Inputs

```javascript
function DatePicker({ value, onChange, label }) {
  const handleDateChange = e => {
    const date = new Date(e.target.value);
    onChange(date);
  };

  return (
    <div className="date-picker">
      <label>{label}</label>
      <input type="date" value={formatDate(value, 'YYYY-MM-DD')} onChange={handleDateChange} />

    </div>
```

### Internationalization Support

```javascript
function LocalizedDate({ date, locale = 'en-US' }) {
  const formatByLocale = (date, locale) => {
    if (locale === 'en-US') {
      return formatDate(date, 'MM/DD/YYYY');
    } else if (locale === 'en-GB') {
      return formatDate(date, 'DD/MM/YYYY');
    } else if (locale === 'ISO') {
      return formatDate(date, 'YYYY-MM-DD');
    }
    return formatDate(date, 'MM/DD/YYYY');
  };

  return <span className="localized-date">{formatByLocale(date, locale)}</span>;
}
```

## Notes

- The function handles Date objects, timestamp numbers, and date strings
- Invalid dates will return "Invalid Date"
- The format patterns are case-sensitive
- For more complex date formatting, consider using libraries like date-fns or moment.js
- Always validate date inputs before formatting
- Consider timezone handling for international applications
