A utility function that formats numbers as currency using the Intl.NumberFormat API with locale support.

<Callout title="Why Use This Utility?">
  - ✅ **Locale Aware** - Supports different currency formats by region - ✅
  **Currency Support** - Works with any ISO currency code - ✅ **Consistent
  Formatting** - Uses browser's built-in Intl API - ✅ **Zero Dependencies** -
  No external libraries required - ✅ **TypeScript Ready** - Full type safety
  included
</Callout>

## API Reference

### Parameters

| Parameter  | Type     | Default   | Description                      |
| ---------- | -------- | --------- | -------------------------------- |
| `amount`   | `number` | Required  | The number to format as currency |
| `currency` | `string` | `"USD"`   | ISO 4217 currency code           |
| `locale`   | `string` | `"en-US"` | BCP 47 locale identifier         |

### Returns

| Type     | Description               |
| -------- | ------------------------- |
| `string` | Formatted currency string |

### Type Definition

```tsx
function formatCurrency(
  amount: number,
  currency?: string,
  locale?: string
): string;
```

## Real-World Examples

### E-commerce Product Listing

```tsx
import { formatCurrency } from "./lib/utils";

interface Product {
  id: string;
  name: string;
  price: number;
  originalPrice?: number;
  currency: string;
}

function ProductGrid({ products }: { products: Product[] }) {
  return (
    <div className="product-grid">
      {products.map((product) => (
        <ProductCard key={product.id} product={product} />

    </div>

function ProductCard({ product }: { product: Product }) {
  const hasDiscount =
    product.originalPrice && product.originalPrice > product.price;
  const discountPercent = hasDiscount
    ? Math.round(
        ((product.originalPrice! - product.price) / product.originalPrice!) *
          100
      )
    : 0;

  return (
    <div className="product-card">
      <h3>{product.name}</h3>

      <div className="price-section">
        <span className="current-price">
          {formatCurrency(product.price, product.currency)}
        </span>

        {hasDiscount && (
          <>
            <span className="original-price">
              {formatCurrency(product.originalPrice!, product.currency)}
            </span>
            <span className="discount-badge">{discountPercent}% OFF</span>
          </>
        )}
      </div>
    </div>
  );
}
```

## Locale Examples

Different locales format the same currency differently:

```tsx
const amount = 1234.56;

// US formatting
formatCurrency(amount, "USD", "en-US"); // "$1,234.56"

// European formatting
formatCurrency(amount, "EUR", "de-DE"); // "1.234,56 €"
formatCurrency(amount, "EUR", "fr-FR"); // "1 234,56 €"

// Indian formatting (lakhs/crores system)
formatCurrency(amount, "INR", "en-IN"); // "₹1,234.56"
formatCurrency(123456, "INR", "en-IN"); // "₹1,23,456.00"
```

## Best Practices

<Callout type="info" title="Formatting Guidelines">
  - Always specify the currency explicitly rather than relying on defaults - Use
  the appropriate locale for your target audience - Consider storing currency
  and locale preferences in user settings - Handle edge cases like zero,
  negative numbers, and very large amounts
</Callout>

### Do's and Don'ts

**✅ Do:**

- Use consistent currency formatting across your app
- Store currency code with monetary values in your data
- Consider user's locale preferences
- Handle decimal precision appropriately for each currency

**❌ Don't:**

- Hard-code currency symbols
- Assume all currencies use the same decimal places
- Ignore locale differences in number formatting
- Mix different currency formats in the same interface

### Performance Considerations

```tsx
// ✅ Good - cache formatter for repeated use
const useCurrencyFormatter = (currency: string, locale: string) => {
  return useMemo(
    () => new Intl.NumberFormat(locale, { style: "currency", currency }),
    [currency, locale]
  );
};

function PriceList({ items, currency, locale }) {
  const formatter = useCurrencyFormatter(currency, locale);

  return (
    <div>
      {items.map((item) => (
        <div key={item.id}>{formatter.format(item.price)}</div>
      ))}
    </div>
  );
}

// ❌ Avoid - creates new formatter each time
function PriceList({ items, currency, locale }) {
  return (
    <div>
      {items.map((item) => (
        <div key={item.id}>{formatCurrency(item.price, currency, locale)}</div>
      ))}
    </div>
  );
}
```


