# capitalize

The `capitalize` utility function capitalizes the first letter of a string while keeping the rest of the string in lowercase. It's useful for formatting user input, display names, and text normalization.

## Usage

```javascript
import { capitalize } from "./utils/capitalize";

const result = capitalize("hello world");
console.log(result); // "Hello world"
```

## API Reference

### Parameters

- `text` (string): The string to capitalize

### Returns

- `string`: The capitalized string

## Examples

### Form Input Formatting

```javascript
function NameInput() {
  const [name, setName] = useState("");

  const handleNameChange = (e) => {
    const formattedName = capitalize(e.target.value);
    setName(formattedName);
  };

  return (
    <input
      type="text"
      value={name}
      onChange={handleNameChange}
      placeholder="Enter your name"
    />
  );
}
```

### User Profile Display

```javascript
function UserProfile({ user }) {
  return (
    <div className="user-profile">
      <h2>
        {capitalize(user.firstName)} {capitalize(user.lastName)}
      </h2>
      <p>Role: {capitalize(user.role)}</p>
      <p>Department: {capitalize(user.department)}</p>
    </div>
  );
}
```

### List Formatting

```javascript
function CategoryList({ categories }) {
  return (
    <ul>
      {categories.map((category) => (
        <li key={category.id}>{capitalize(category.name)}</li>
      ))}
    </ul>
  );
}
```

### Search Results

function SearchResults({ results, query }) {
  return (
    <div>

### Status Indicators

```javascript
function OrderStatus({ status }) {
  const getStatusColor = (status) => {
    switch (status.toLowerCase()) {
      case "pending":
        return "orange";
      case "processing":
        return "blue";
      case "completed":
        return "green";
      case "cancelled":
        return "red";
      default:
        return "gray";
    }
  };

  return (
    <span className="status-badge" style={{ color: getStatusColor(status) }}>
      {capitalize(status)}
    </span>
  );
}
```

### Form Validation Messages

```javascript
function ValidationMessage({ field, error }) {
  if (!error) return null;

  return (
    <div className="error-message">
      {capitalize(field)} {error}
    </div>
  );
}

// Usage
<ValidationMessage field="email" error="is required" />;
// Renders: "Email is required"
```

### Dynamic Content

```javascript
function WelcomeMessage({ timeOfDay, userName }) {
  return (
    <div className="welcome">
      <h2>
        Good {capitalize(timeOfDay)}, {capitalize(userName)}!
      </h2>
    </div>
  );
}

// Usage
<WelcomeMessage timeOfDay="morning" userName="john doe" />;
// Renders: "Good Morning, John doe!"
```

## Notes

- The function converts the entire string to lowercase first, then capitalizes the first character
- Empty strings return empty strings
- Non-string inputs should be converted to strings first
- For title case (capitalizing each word), consider using a more sophisticated function
- Works well with other string manipulation utilities
