# useToggle

The `useToggle` hook provides a simple way to manage boolean state with toggle functionality. It's perfect for handling UI states like modals, dropdowns, or any on/off functionality.

## Usage

```tsx
import { useToggle } from "./hooks/useToggle";

function Modal() {
  const [isOpen, toggleOpen] = useToggle(false);

  return (
    <div>
      <button onClick={toggleOpen}>{isOpen ? "Close" : "Open"} Modal</button>
      {isOpen && (
        <div className="modal">
          <p>Modal content here!</p>
          <button onClick={toggleOpen}>Close</button>
        </div>
      )}
    </div>
  );
}
```

## API Reference

### Parameters

- `initialValue` (boolean, optional): The initial boolean value. Defaults to `false`.

### Returns

- `[value, toggle]` (tuple):
  - `value` (boolean): The current boolean state
  - `toggle` (function): Function to toggle the state or set it to a specific value

### Loading States

```tsx
function DataFetcher() {
  const [isLoading, toggleLoading] = useToggle(false);
  const [data, setData] = useState(null);

  const fetchData = async () => {
    toggleLoading(true);
    try {
      const response = await fetch("/api/data");
      const result = await response.json();
      setData(result);
    } catch (error) {
      console.error("Failed to fetch data:", error);
    } finally {
      toggleLoading(false);
    }
  };

  return (
    <div>
      <button onClick={fetchData} disabled={isLoading}>
        {isLoading ? "Loading..." : "Fetch Data"}
      </button>
      {data && (
        <div>
          <pre>{JSON.stringify(data, null, 2)}</pre>
        </div>
      )}
    </div>
  );
}
```

### Advanced Toggle with Specific Values

```tsx
function AdvancedToggle() {
  const [isEnabled, toggle] = useToggle(true);

  return (
    <div>
      <p>Feature is {isEnabled ? "enabled" : "disabled"}</p>

      <button onClick={() => toggle()}>Toggle</button>
      <button onClick={() => toggle(true)}>Enable</button>
      <button onClick={() => toggle(false)}>Disable</button>
    </div>
  );
}
```

## Notes

- The toggle function can be called without arguments to flip the current state
- You can also pass a specific boolean value to set the state directly
- Perfect for managing UI states that have only two possible values
- Works well with other hooks for complex state management scenarios
