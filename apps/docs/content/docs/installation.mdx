# Installation

<Callout title="Quick Start">
  react-usekit can be installed globally, used with NPX, or added to your
  project. Choose the method that works best for your workflow.
</Callout>

## Installation Options

<Tabs items={['Global (Recommended)', 'NPX (One-time)', 'Project-specific']}>
<Tab value="Global (Recommended)">
### Global Installation

Best for developers who work on multiple React projects.

```bash
npm install -g react-usekit
```

**Usage:**

```bash
cd your-react-project
react-usekit init
```

**Pros:**

- ✅ Use in any React project
- ✅ Always available
- ✅ Easy to update globally
- ✅ Fastest execution

**Cons:**

- ❌ Requires global installation
- ❌ May have version conflicts

</Tab>

<Tab value="NPX (One-time)">
### NPX Usage

Perfect for occasional use or trying out react-usekit.

```bash
cd your-react-project
npx react-usekit init
```

**Pros:**

- ✅ No installation required
- ✅ Always uses latest version
- ✅ No global dependencies
- ✅ Great for CI/CD

**Cons:**

- ❌ Slower first run (downloads package)
- ❌ Downloads package each time

</Tab>

<Tab value="Project-specific">
### Project Installation

Add react-usekit as a development dependency to your project.

```bash
npm install --save-dev react-usekit
```

**Add to package.json scripts:**

```json
{
  "scripts": {
    "add-hooks": "react-usekit init",
    "add-utils": "react-usekit init"
  }
}
```

**Usage:**

```bash
npm run add-hooks
```

**Pros:**

- ✅ Version locked to project
- ✅ Consistent across team
- ✅ Works in CI/CD
- ✅ No global installation

**Cons:**

- ❌ Must install per project
- ❌ Takes up space in node_modules

</Tab>
</Tabs>

## Setup Guide

<Steps>
<Step>
### 1. Navigate to Your React Project

```bash
cd your-react-project
```

<Callout type="info">
Make sure you're in a React project directory with a `package.json` file.
</Callout>
</Step>

<Step>
### 2. Run the Interactive CLI

```bash
react-usekit init
```

The CLI will guide you through:

- **Type Selection**: Choose between hooks or utilities
- **Language**: TypeScript or JavaScript
- **Directory**: Where to place the files
- **Template Selection**: Multi-select the items you want
- **Confirmation**: Review before generating

</Step>

<Step>
### 3. Start Using Your Templates

Once generated, import and use your new hooks/utilities:

```tsx
// Import your new hook
import { useLocalStorage } from "./src/hooks/useLocalStorage";

function App() {
  const [name, setName] = useLocalStorage("userName", "");

  return (
    <input
      value={name}
      onChange={(e) => setName(e.target.value)}
      placeholder="Your name will be saved automatically"
    />
  );
}
```

</Step>
</Steps>

## Troubleshooting

### Common Issues

**"Command not found" error:**

```bash
# If using global installation
npm install -g react-usekit

# If using NPX
npx react-usekit init
```

**Permission errors on macOS/Linux:**

```bash
# Use sudo for global installation
sudo npm install -g react-usekit

# Or use NPX (no permissions needed)
npx react-usekit init
```

**TypeScript errors:**

- Ensure TypeScript 4.5+ is installed
- Check your `tsconfig.json` includes the generated files
- Verify React types are installed: `npm install --save-dev @types/react`
