## Hook Categories

### üè™ State Management

Hooks for managing and persisting component state.

- **useLocalStorage** - Sync state with localStorage
- **useToggle** - Boolean state with convenience methods

### üì° Async & Data

Hooks for handling asynchronous operations and data fetching.

- **useAsync** - Complete async operation management
- **useDebounce** - Delay expensive operations

### üéØ UI & Interaction

Hooks for user interface and interaction patterns.

- **useClickOutside** - Modal and dropdown patterns
- **useEventListener** - Custom event handling
- **useWindowSize** - Responsive design patterns

### ‚è∞ Timing & Effects

Hooks for timing-based operations and side effects.

- **useInterval** - Declarative intervals with controls

## Quick Examples

### Simple State with Persistence

```tsx
import { useLocalStorage } from "./hooks/useLocalStorage";

function UserPreferences() {
  const [theme, setTheme] = useLocalStorage("theme", "light");

  return (
    <button onClick={() => setTheme(theme === "light" ? "dark" : "light")}>
      Current theme: {theme}
    </button>
  );
}
```

### API Data Fetching

```tsx
import { useAsync } from "./hooks/useAsync";

function UserProfile({ userId }) {
  const {
    data: user,
    loading,
    error,
  } = useAsync(() => fetch(`/api/users/${userId}`).then((res) => res.json()), {
    immediate: true,
  });

  if (loading) return <div>Loading...</div>;
  if (error) return <div>Error: {error.message}</div>;
  return <div>Hello, {user?.name}!</div>;
}
```

### Modal with Click Outside

```tsx
import { useToggle } from "./hooks/useToggle";
import { useClickOutside } from "./hooks/useClickOutside";

function Modal() {
  const [isOpen, { toggle, setFalse }] = useToggle();
  const modalRef = useClickOutside(setFalse);

  return (
    <>
      <button onClick={toggle}>Open Modal</button>
      {isOpen && (
        <div className="modal-overlay">
          <div ref={modalRef} className="modal">
            <h2>Modal Content</h2>
            <button onClick={setFalse}>Close</button>
          </div>
        </div>
      )}
    </>
  );
}
```

## Best Practices

<Callout type="info" title="Hook Guidelines">
  - Always follow the [Rules of
  Hooks](https://react.dev/reference/rules/rules-of-hooks) - Use TypeScript for
  better development experience - Handle loading and error states in async
  operations - Clean up side effects properly (most hooks handle this
  automatically)
</Callout>

### Performance Tips

1. **useDebounce** - Use for expensive operations like API calls
2. **useCallback** - Combine with hooks that accept callback dependencies
3. **useMemo** - Memoize expensive computations that depend on hook values
4. **useWindowSize** - Use debouncing to avoid excessive re-renders

### Common Patterns

```tsx
// Combine hooks for powerful patterns
function SearchComponent() {
  const [query, setQuery] = useLocalStorage("searchQuery", "");
  const debouncedQuery = useDebounce(query, 300);

  const { data: results, loading } = useAsync(() => searchAPI(debouncedQuery), {
    immediate: debouncedQuery.length > 2,
  });

  return (
    <div>
      <input
        value={query}
        onChange={(e) => setQuery(e.target.value)}
        placeholder="Search..."
      />
      {loading && <div>Searching...</div>}
      {results?.map((item) => (
        <div key={item.id}>{item.title}</div>
      ))}
    </div>
  );
}
```
